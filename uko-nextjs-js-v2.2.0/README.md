- **22개 컴포넌트**: 완전한 React 컴포넌트 라이브러리
- **자동 Figma 토큰 동기화**: 디자인 토큰이 자동으로 동기화됩니다
- **인터랙티브 문서**: 브라우저에서 바로 확인 가능한 컴포넌트 문서
